# 面向端侧大模型低比特量化推理的近似矩阵乘法加速单元设计

## 项目简介
本项目旨在为边缘计算设备（端侧）上的大型语言模型（LLM）设计一种高效的硬件加速单元。通过结合**低比特量化技术**与**近似矩阵乘法（Approximate Matrix Multiplication）**，在保证模型推理精度的前提下，大幅降低硬件的功耗、面积以及计算延迟。

## 技术栈与目录结构
本项目包含算法验证与硬件实现两大部分：

* **算法端 (PyTorch)**
    * 大模型低比特量化算法的软件模拟与精度评估。
    * 近似计算误差的建模与补偿机制。
* **硬件端 (Scala / SpinalHDL / Vivado)**
    * 基于 SpinalHDL 的近似乘加器（MAC）与脉动阵列（Systolic Array）微架构设计。
    * RTL 代码生成与功能仿真。
    * 基于 Vivado 的逻辑综合、功耗与面积（PPA）评估。
